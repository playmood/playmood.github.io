gcc安全编译选项


playmood   May 18, 2020   pwn  

各种安全选择的编译参数如下：

  • NX：-z execstack / -z noexecstack (关闭 / 开启)
  • Canary：-fno-stack-protector /-fstack-protector / -fstack-protector-all (关闭 / 开启 / 全开启)
  • PIE：-no-pie / -pie (关闭 / 开启)
  • RELRO：-z norelro / -z lazy / -z now (关闭 / 部分开启 / 完全开启)
